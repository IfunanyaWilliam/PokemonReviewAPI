PokemonReviewAPI

This a project for Pokemon lovers. Pokemon can be given ratings by admirers.


The project has most of the functionalities needed such:

=> Adding Pokemon
=> Adding a review
=> Updating and deleting pokemon, review, owner country. 


Repository pattern is adopted for this project.


Teck stack
It is implemented with C#, ASP.NET, SQL Server.


Security:
=> Implemented authentication with Jwt. 
=> Implemented Jwt refresh token
